A beginner-friendly project demonstrating how a Reinforcement Learning agent, built with PyTorch, learns to play the classic Snake game from scratch.

This repository provides a clear, modular, and well-commented implementation of a Deep Q-Learning (DQN) agent designed to master the Snake game. It's the perfect starting point for anyone looking to see a practical application of RL in action.

## ✨ Key Features

- **Deep Q-Learning from Scratch:** A pure Python and PyTorch implementation of a DQN agent.
- **Modular Codebase:** The project is logically separated into distinct files for the agent, the game environment, the neural network model, and the training/playing scripts.
- **Clean Game Interface:** The game is built with Pygame and features a modern, clean visual style.
- **Live Training Visualization:** Watch the agent's score and mean score evolve in real-time with a live plot generated by `matplotlib`.
- **Pre-trained Model Included:** Don't want to wait for training? A pre-trained model (`model.pth`) is included in the `model/` directory, ready to be used.
- **Experience Replay:** The agent uses a `deque` as a memory buffer to store past experiences and sample from them, stabilizing the training process.

## 🤖 How It Works: The Reinforcement Learning Approach

This project uses a technique called **Deep Q-Learning (DQN)**. The goal is to train a "brain" (our neural network) that can look at the current game situation (the **state**) and decide the best action to take.

### 1. The State (What the AI "Sees")

The AI doesn't see the screen like we do. Instead, we feed it a carefully designed array of 11 boolean (`True`/`False`) values that describe its immediate surroundings.

The 11 values represent:
- **[Danger Straight]**: Is there a wall or its own body immediately in front of the snake?
- **[Danger Right]**: Is there danger if the snake turns right relative to its current direction?
- **[Danger Left]**: Is there danger if the snake turns left relative to its current direction?
- **[Move Direction Left]**: Is the snake currently moving left?
- **[Move Direction Right]**: Is the snake currently moving right?
- **[Move Direction Up]**: Is the snake currently moving up?
- **[Move Direction Down]**: Is the snake currently moving down?
- **[Food Left]**: Is the food to the left of the snake's head?
- **[Food Right]**: Is the food to the right of the snake's head?
- **[Food Up]**: Is the food above the snake's head?
- **[Food Down]**: Is the food below the snake's head?

This 11-item list is the **state** that gets fed into the neural network.

### 2. The Action (What the AI "Does")

The AI has three possible moves relative to its current direction:
1.  **Continue Straight**
2.  **Turn Right**
3.  **Turn Left**

The neural network outputs a value for each of these three actions. The agent then picks the action with the highest value.

### 3. The Reward (How the AI "Learns")

The agent is "rewarded" or "punished" based on its actions:
- **Eat Food:** `+10` points (Good!)
- **Game Over (Collision):** `-10` points (Bad!)
- **Any other move:** `0` points (Neutral)

The agent's goal is to **maximize its total reward**. Over thousands of games, it learns that eating food leads to a positive reward and hitting walls leads to a negative one, so it adjusts its strategy accordingly. This process of trial and error is the essence of Reinforcement Learning.

## 📂 Project Structure

The code is organized to be easy to understand and navigate.

```
PyTorch-Snake-AI/
├── model/
│   └── model [Pre-Trained on 200 games].pth     # A pre-trained model file (You have to rename it into "model.pth")
├── agent.py                                     # Defines the Agent, its memory, and learning logic
├── game_ai.py                                   # The Snake game environment (built with Pygame)
├── model.py                                     # The PyTorch neural network (Linear_QNet) and trainer
├── play.py                                      # Script to run and watch the pre-trained agent play
└── train.py                                     # Script to train the agent from scratch
```

-   **`agent.py`**: The core of the AI. It gets the game state, decides on an action, and stores experiences in its memory.
-   **`game_ai.py`**: The environment. It handles all game logic: moving the snake, placing food, detecting collisions, and updating the display.
-   **`model.py`**: Defines the `Linear_QNet` neural network architecture and the `QTrainer` which implements the Bellman equation to update the model's weights.
-   **`train.py`**: The main script you run to start the training process. It orchestrates the interaction between the agent and the game.
-   **`play.py`**: A simple script to load the `model.pth` and see your trained agent play the game without any further training or random moves.

## 🛠️ Setup and Installation

Follow these steps to get the project running on your local machine.

#### Prerequisites
- **Python 3.8+**
- **Git**

#### 1. Clone the Repository
```bash
git clone https://github.com/rudra-mondal/PyTorch-Snake-AI.git
cd PyTorch-Snake-AI
```

#### 2. Create a Virtual Environment (Recommended)
This keeps the project's dependencies isolated from your system's Python installation.
```bash
# On macOS and Linux
python3 -m venv venv
source venv/bin/activate

# On Windows
python -m venv venv
.\venv\Scripts\activate
```

#### 3. Install Dependencies
Install all required libraries using the `requirements.txt` file.
```bash
pip install -r requirements.txt
```

## 🚀 Usage

You can either train a new model from scratch or use the pre-trained model to see the agent play.

### 1. To Train a New Agent
Run the `train.py` script. This will open two windows:
- A **Pygame window** where you can see the snake playing the game.
- A **Matplotlib window** that plots the agent's score and average score over time.

```bash
python train.py
```
The model will automatically save itself in the `model/` directory whenever it achieves a new high score. Let it run for at least a few hundred games to see significant improvement.

### 2. To Watch the Pre-Trained Agent Play
Run the `play.py` script. This will load the `model/model.pth` file and you will see the trained agent play the game at full speed.

```bash
python play.py
```
To quit, simply close the Pygame window.
