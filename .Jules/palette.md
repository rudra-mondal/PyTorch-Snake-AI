## 2024-05-24 - Add Grid Background to Snake Game
**Learning:** Visual guides for discrete spatial movement makes it much easier to comprehend block-based grid movements like the Snake Game, without getting visually confused about positions.
**Action:** Always consider drawing a grid background when an interface uses discrete block-based spatial positioning to provide clear visual boundaries for elements.

## 2024-05-24 - Directional Indicators for AI Movement
**Learning:** For AI systems that make discrete moves or turns (like a snake game), showing visually which way the entity is currently "facing" helps users intuit the next move before it happens, removing visual ambiguity about the head/tail or direction of travel.
**Action:** Always consider adding "eyes" or directional arrows to autonomous moving entities so users can easily perceive their orientation.
